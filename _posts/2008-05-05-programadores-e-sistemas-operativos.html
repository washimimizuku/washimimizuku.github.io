--- layout: post title: Programadores e Sistemas Operativos --- Há já 8 anos que ando nisto da programação (mais, se contarmos com o tempo antes de ter começado a trabalhar), e a maioria desse tempo trabalhei em PHP, Perl, e Python. Também sempre usei o Linux (e também o Mac OS X) para trabalhar com essas linguagens, o que a mim me parece tão óbvio, que nem carece de explicação.

Foi preciso vir para a Suíça para descobrir que nem todos os programadores de PHP usam Linux ou Mac OS X para trabalhar. Na realidade, aqui, a maioria usa Windows. Não percebo. Eu percebo que programadores de Java utilizem Windows, e os de .net é óbvio que também o façam. Agora programadores de PHP?

Elucidem-me. Será que aí em Portugal tive a sorte de estar em ambientes em que todos usavam algum tipo de UNIX para trabalhar? Será que existem muitos programadores de PHP / Perl / Python em Portugal a usar Windows e eu nunca tinha reparado? E se existem, porquê? O que raio os leva a usar Windows para programar?
